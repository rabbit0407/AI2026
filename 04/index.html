웹 프론트엔드의 심장

모든 웹 페이지는 이 세 가지 기술의 조합으로 만들어집니다.

HTML5

웹의 '뼈대'를 담당합니다. 텍스트, 이미지, 버튼 등 요소들의 의미와 구조를 정의하는 마크업 언어입니다.

Structure

CSS3

웹의 '근육과 피부'입니다. 레이아웃, 색상, 애니메이션 등 시각적인 스타일을 입혀 사용자 경험을 극대화합니다.

Style

JavaScript

웹의 '두뇌'입니다. 클릭 이벤트, 데이터 통신, 동적 화면 업데이트 등 상호작용이 가능한 지능적인 웹을 만듭니다.

Logic

How Web Works?

사용자가 URL을 입력하면 브라우저는 DNS를 통해 서버의 IP 주소를 찾고, HTTP 프로토콜을 이용해 리소스를 요청합니다. 서버는 이에 응답하여 데이터를 보내주고 브라우저는 이를 렌더링합니다.

  • ✔
    Client-Server 모델 요청하는 클라이언트(브라우저)와 응답하는 서버
  • ✔
    HTTP/HTTPS 프로토콜 데이터를 주고받기 위한 통신 규약
  • ✔
    API & Database 데이터를 처리하고 영구적으로 저장하는 시스템
